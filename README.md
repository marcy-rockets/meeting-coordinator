# Global Meeting Coordinator

時差やサマータイムを考慮して、世界各地の会議時間を簡単に調整・共有できるWebアプリです。

## 特徴
- **正確な時差計算**: Luxonライブラリを使用し、サマータイムも考慮。
- **マルチ言語対応**: 日本語と英語をワンクリックで切り替え。
- **チャット共有**: SlackやTeamsにそのまま貼れる形式でテキストを出力。
- **モダンUI**: グラスモーフィズムを採用したプレミアムなデザイン。

## デプロイ方法
このリポジトリは静的サイトとして構成されています。GitHubにアップロードした後、Vercel等のサービスに接続するだけで公開可能です。
